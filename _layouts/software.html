Local Option:
  The goal of the Local Option software project was to help everyday users free large photo albums from disparate walled garden ecosystems, and easily manage them in a central offline location. The technical approach was to facilitate ETL from common photo album providers into a standalone program having GUI-based CRUD features. The original description + posts below cover background, MVP, backend and wireframe - plus reasons for stopping development. Please direct any questions or comments here!
