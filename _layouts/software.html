Local Option:
  Local Option was a software project built to decouple photo albums from proprietary ecosystems (walled gardens). Our goal was partly an ETL tool and partly a very user-friendly, GUI-driven CRUD tool powered by web-based Computer Vision. The simple requirements were:
  • Offline-first
  • Mac, Windows, iOS, Android
  • Slick UI
  • Base level of feature parity/object detection

Sadly, Local Option met the same fate as many partner-passion-projects. Aside from typical time constraint/career demands, we ran into blockers on:

  • Offline requirement X any object detection other than face
  • Cross-platform requirement X container approach
  • Slick UI requirement x Electron performance
  • Feature parity requirement X newness of OpenCV.js and WebAssembly (~2017)

Tears aside, Local Option was a great intro to Computer Vision for two people whose previous "AI" experience was limited to NLP. We could have certainly finished it in a "hacky" manner, but decided to nix the spaghetti code/Rube Goldberg approach and table it for the time being...Check out the project below, and please direct any questions or comments here!
